## Description

Flighter App from The Guardian News API

Built using:

-   React
-   Redux
-   Redux-Saga
-   Redux-Forms
-   Redux-Persist
-   Material UI
-   Axios

## Usage:

To run locally follow these instructions:

### Installation

```bash
 npm install
```

### Running the app

```bash
npm run start
```

### Test

```bash
npm run test
```
